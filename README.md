# PSO2DamageDump
Re-implementation of the original PSO2DamageDump, updated for SL. Compatible with Overparse.

## How to use
Drop the pre-compiled PSO2DamageDump.dll (either MinGW or MSVC should work) in the Plugins folder. Compatible with the original pso2h.dll and my reverse engineered pso2h.dll.
Once the plugin is loaded, it will output Overparse formatted CSV files in _pso2_bin/damagelog_.

Go get Overparse [here!](https://github.com/mysterious64/OverParse/)
